# Notes Taking App 

* ###   To read a note's body
```
  node app.js read --tittle="Tittle of Note"`
```
 
* ###  To list all notes tittle
```
  node app.js list 
```
 
* ###  To add a note 
```
  node app.js add --tittle="Tittle of Note" --body="Body of Note"
```
 
* ###  To remove a note 
```
  node app.js remove --tittle="Tittle of Note"
```
